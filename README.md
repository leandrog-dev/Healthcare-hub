# HealthCare Hub

Aplicação web desenvolvida como projeto acadêmico com foco em saúde e bem-estar.

---

## 📌 Sobre o projeto

O HealthCare Hub é uma aplicação web criada para auxiliar usuários no cuidado com a saúde, oferecendo funcionalidades como:

- Cálculo de IMC
- Dicas de saúde
- Agendamento
- Quiz interativo

O projeto foi desenvolvido com foco em uma navegação simples, interface intuitiva e boa experiência do usuário.
